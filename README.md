## Requirements

- [Node.js](https://nodejs.org/en/download/package-manager/)
- [TypeScript](https://www.typescriptlang.org/)
- [Lerna](https://lerna.js.org/)

---

## Istallation

```sh
yarn
```

then

```sh
yarn bootstrap
```

## Running

```sh
yarn start
```

### Build

Build app:

```sh
yarn build:app
```
